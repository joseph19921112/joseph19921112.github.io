arch安装 


Archlinux
=====================================================
1 测试网络
    wifi连接
    # wifi-menu
    # ping archlinux.org
------------------------------
2 选择镜像源
    备份
    # cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.bak
    # nano /etc/pacman.d/mirrorlist
    修改方法 1 
        注销全部
        Server = http://mirrors.aliyun.com/archlinux/$repo/os/$arch
    修改方法 2
        进入编辑器，ctrl+w 查找 China，去掉 ##，执行ctrl+x退出保存
    更新
    # pacman -Syy
------------------------------
3 分区（可以提前分好）
    查看硬盘
    # lsblk
    进入 fdisk 功能模块
    # fdisk /dev/sda
    MBR分区
        - n : 添加一个分区
        - p : 主分区
        - 分区编号：1，2，3等
        - 分区开始位置：（默认）
        - 结束位置：（+ xxx M/G）
        - w : 保存
    GPT分区
        - g : 建立gpt分区表
        - n : 添加一个分区
        ESP分区：500M（最少）
        - p : 查看一下
        - w : 保存
------------------------------
4 格式化分区
    格式化交换分区
    # mkswap /dev/sda3
    激活交换分区
    # swapon /dev/sda3
    格式化ESP分区
    # mkfs.vfat /dev/sda1
    格式化ext4分区
    # mkfs.ext4 /dev/sda2
------------------------------
5 挂载分区
    挂载根
    # mount /dev/sda1 /mnt
    根目录下，创建boot文件夹、efi文件夹
    # mkdir -p /mnt/boot/efi 
    挂载根efi
    # mount /dev/sda1 /mnt/boot/efi
------------------------------
6 安装系统
    联网下载 base base-devel
    # pacstrap -i /mnt base base-devel
    配置fstab文件
    # genfstab -U /mnt >> /mnt/etc/fstab
    检查一下
    # cat /mnt/etc/fstab
------------------------------
7 进入新系统的root用户
    切换用户
    # arch-chroot /mnt
    设置系统可用语言
    # nano /etc/locale.gen
    查找
        en_US.UTF-8 UTF-8 
        zh_CN.UTF-8 UTF-8
    执行设定好语言
    # locale-gen
    设置系统语言，默认英语
    # echo LANG=zh_CN.UTF-8 > /etc/locale.conf
    安装Dejavu 和 文泉驿等宽微米黑字体
    # pacman -S ttf-dejavu wqy-microhei
------------------------------
8 创建用户
    root用户密码
    # passwd
    创建个人用户名
    # useradd -m -g users -s /bin/bash 用户名
    设置个人用户密码
    # passwd 用户名
    个人用户添加sudo权限
    # nano /etc/sudoers
    在 root ALL=(ALL) ALL 下面，添加: 用户名 ALL=(ALL) ALL
------------------------------
9 安装grub引导
    grub-bios 模式
    # pacman -S grub
    # grub-install /dev/sda 
    # grub-mkconfig -o /boot/grub/grub.cfg
    grub-efi 模式(不完善)
    # pacman -S grub efibootmgr os-prober
    # grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=arch_grub --recheck
    # grub-mkconfig -o /boot/grub/grub.cfg
    # cp /boot/grub/grub.cfg.new /boot/grub/grub.cfg
------------------------------
10 安装网络
    安装有线: 只需要启动dhcp
    # systemctl enable dhcpcd
    安装无线: 安装iw、wpa支持
    # pacman -S iw
    # pacman -S wpa_supplicant
    安装图形化网络管理器
    # pacman -S networkmanager
------------------------------
11 安装显卡
    确定显卡型号
    # lspci | grep VGA
    安装显卡
    # pacman -S 驱动包
    官方仓库提供的驱动包:
        vesa   : xf86-video-vesa
        intel  : xf86-video-intel
        vmware :
        # pacman -S xf86-video-vmware
        # pacman -S xf86-input-vmmouse  

    安装 xorg
    # pacman -S xorg
------------------------------
12 安卓桌面
    安装 deepin
    # pacman -S deepin deepin-extra
    安装 mate
    # pacman -S mate mate-extra
    安装 xfce4
    # pacman -S xfce4 xfce4-goodies
    安装 lxde（包含lxdm）
    # pacman -S lxde
------------------------------
13 安装启动器
    安装 lxdm
    # pacman -S lxdm
    启动登入管理器
    # systemctl start lxdm.service
    开机自启动登入管理器
    # systemctl enable lxdm.service
------------------------------
14 重启
    退出chroot进入iso
    # exit
    重启
    # reboot
------------------------------
15 其他优化
    安装 open-vm-tools
    # pacman -S open-vm-tools
    # pacman -S gtkmm3
    # systemctl enablevmtoolsd.service
    # systemctl enable vmware-vmblock-fuse.service
    压缩软件
    # pacman -S fille-roller 
    或者 # pacman -S fille-roller xarchive
    安装系统对话框
    # pacman -S dialog
    添加archlinuxcn源
    # nano /etc/pacman.conf
    文件末尾添加
    [archlinuxcn]
    SigLevel = Never
    Server = http://mirrors.163.com/archlinux-cn/$arch
    更新源
    # pacman -Syy
    立刻启动网络
    # systemctl start dhcpcd
    # systemctl start NetworkManager
    开机自动启动网络
    # systemctl enable dhcpcd
    # systemctl enable NetworkManager
    安装触摸板驱动
    # pacman -S xf86-input-synaptics

    桌面选择
    > 窗口管理器(WM)
        - Compiz
        - openbox
        - enlightenment
        - fvwm
    > 登陆器(DM)
        - Entrance：基于 EFL 的显示管理器，高度实验性质。
        - GDM：GNOME 显示管理器。
        - LXDM：LXDE 显示管理器，可以独立于 LXDE 桌面环境使用。
        - LightDM：跨桌面显示管理器，可以使用任何工具集写就的各种前端。
        - MDM：MDM 显示管理器，GDM2的一个 fork，Linux Mint 中使用。
        - SDDM：基于 QML 的显示管理器，KDE 4 的 kdm 的继任者。
        - XDM：带有 XDMCP，主机选择支持的 X 显示管理器。
        - SLiM：轻量且优雅的图形化登录解决方案（不再继续开发）。
        - KDM：KDE4 显示管理器（不再继续开发）。
    > 桌面(DE)
        - GNOME(gdm)
        - KDE(sddm)
        - Cinnamon
        - LXDE(lxdm)
        - Xfce
        - DDE(Lightdm)
        - MATE
        - fvwm
