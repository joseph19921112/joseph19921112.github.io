E - 类 - 9 - 静态方法 


静态方法
            1 装饰器 @staticmethod
            2 不能写 self
            1 z主要用于某个方法不访问：实例属性、实例方、类属性、类方法
        调用：
            1 通过 实例调用
            2 通过 类名调用  (推荐的方式)
        $Exp:
            $$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$
            $  class EncodeUtils:
            $      @staticmethod
            $      def encode(data, charset):
            $          print("加密.....")
            $      @staticmethod
            $      def decode(data, charset):
            $          print("解密.....")
            $  eu = EncodeUtils()
            $  eu.encode("hello", "UTF-8")  # 加密.....
            $  eu.decode("hello", "UTF-8")  # 解密.....
            $$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$$
